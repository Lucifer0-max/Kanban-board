Kanban Board

A simple and interactive Kanban Board built using HTML, CSS, and JavaScript. This project helps users organize tasks by moving them across different stages of a workflow using a clean and responsive interface.

Features

Create new tasks
Drag and drop tasks between columns
Edit task content
Delete tasks
Responsive design
Pure JavaScript implementation (no frameworks)

Technologies Used

HTML5
CSS3
JavaScript (ES6)

Project Structure

kanban-board/
│
├── index.html      # Main HTML file
├── style.css       # Styling and layout
├── script.js       # Application logic
└── README.md

Getting Started

Clone the repository:
git clone https://github.com/your-username/kanban-board.git
Navigate to the project folder:
cd kanban-board
Open index.html in your browser.

No additional dependencies or installation steps are required.

How It Works

Users can add tasks to the board.
Tasks can be moved between columns using drag and drop.
Changes are automatically saved in Local Storage.
Task data remains available even after page refreshes or browser restarts.
